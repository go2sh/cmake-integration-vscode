# Installation

# Requirements

To use the extensions, you need CMake 3.7 or higher installed at your computer.

# Installation

The installation of the CMake Integration Extension is easy. Just open Visual Studio and click on the extension icon. Search for go2sh.cmake-integration and press the Install button.

WARNING

This extension is incompatible with other CMake Extensions for VS Code.
